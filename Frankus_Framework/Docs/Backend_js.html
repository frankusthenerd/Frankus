cFile

constructor(name)


Creates a file module.
name The name of the file.

Read()


Reads the contents of the file.

Read_Binary()


Reads binary data. Only the buffer is populated.

Read_Stream(write_stream)


Streams the file to a write stream to prevent memory exhaustion.
write_stream The write stream to push the data to.

Write()


Writes the contents of a file.

Write_From_Data()


Writes the file from the data.

Write_Binary()


Writes binary data.

Write_From_Buffer()


Writes the data from the buffer.

Add(line)


Adds a line to the file.
line The line to add.

Add_Lines(lines)


Adds a bunch of lines to the file.
lines The list of lines to add.

Add_Object(object)


Adds an object to the file.
object The object to add to the file.

Remove(index)


Removes a line at a specified index.
index The index of the line to remove.
throws An error if the index is not valid.

Count()


Gets the number of lines in the file.
returns The number of lines in the file.

Get_Line_At(index)


Gets the string at the index.
index The index of the string.
returns The string at the index.
throws An error if the string is not present.

Get_Line()


Gets a line from the file sequentially.
returns The read line.
throws An error if no more lines can be read.

Get_Number()


Reads a numeric value from an index.
returns The number read.
throws An error if the number could not be read.

Get_Object(object)


Reads an object from the file.
object The object to read in.
throws An error if the object could not be read.

Clear()


Clears out the file's lines.

Has_More_Lines()


Determines if a file has more lines.
returns True if there are more lines, false otherwise.

Sort_Lines(ascending, numeric)


Sorts the lines according to the order.
ascending If true then the lines are sorted ascending.
numeric If true then the lines are sorted as numbers.

Mod_Data(on_mod)


Modifies lines of data.
on_mod Called when lines needs modding. Passed in line, return modded line.

Get_Extension(file)


Gets the extension of the given file.
file The file path.
returns The file extension without the dot.

Get_File_Name(file)


Gets the name of a file.
file The file to get the name of.
returns The name of the file.

Get_File_Title(file)


Gets the title of the file.
file The file to get the title of.
returns The title of the file.

Get_Local_Path(folder)


Gets the local path given the folder.
folder The folder path.
returns The platform depend OS path.

Escape_Path(folder)


Escapes a folder path to platform independent path separators.
folder The folder path.
returns The path that is platform independent.

Create_Folder(folder)


Creates a new folder.
folder The folder to create.

Query_Files(folder, search)


Queries a group of files in a directory or a group of folders.
folder The folder to look for files.
search The search string. Used Frankus wildcards.
returns The list of files in the folder.

Get_File_And_Folder_List(folder, exclude)


Grabs the list of files from the folder including the subfolders.
folder The folder to look in.
exclude An optional parameter to exclude a list of folders.
returns The list of files and folders.

Change_Folder(folder)


Changes to a specific folder.
folder The folder to change to.

Revert_Folder()


Reverts back to the original startup folder.

Get_File_Modified_Time(file)


Gets the modification time of a file.
file The file to get the modification time for.
returns The modification time of the file.

To_URL_Path(file)


Converts a path to a URL path.
file The file to convert.
returns The URL file path.

Does_File_Exist(file)


Checks to see if a file exists.
file The file to check.
returns True if the file exists, false otherwise.

Copy_File(source, dest)


Copies a file from source to dest.
source The source file to copy.
dest The destination file.

Get_File_Size(file)


Gets the size of a file.
file The file to get the size for.
returns The file size in bytes.

Delete_File(file)


Deletes a file.
file The name of the file to delete.

Is_File_Name(file)


Determines if a file name is a file.
file The file to test.
returns True if it is a file name, false otherwise.
