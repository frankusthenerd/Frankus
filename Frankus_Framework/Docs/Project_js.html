cProject

constructor(project)


Creates a new project tool.
name The name of the project to manage.

Zip(on_zip)


Creates a zip archive of project.
folder The folder to archive.
on_zip Called when the project is zipped.

Zip_Cpp_Project(on_zip)


Creates a zip archive of a C++ project.
folder The folder to archive.
on_zip Called when the project is zipped.

Zip_JavaScript_Project(on_zip)


Creates a zip archive of a C++ project.
folder The folder to archive.
on_zip Called when the project is zipped.

Zip_Editor(on_zip)


Zips up the editor for use as standalone program.
on_zip Called when the package is created.

Archive_Site(on_archive)


Archives the entire site. Drops the file in the download folder.
on_archive Called when site is archived.

Upload_File(name, on_upload, on_error)


Uploads a file to a server.
name The name of the file to upload.
on_upload Called when the file is uploaded.
on_error Called if there was an error. A message is passed in.

Download_File(name, on_download, on_error)


Downloads a file from the server.
name The name of the file to download.
on_download Called when the file is downloaded.
on_error Called if there was an error. A message is passed in.

Create_Folder(folder, on_create, on_error)


Creates a folder on the server.
folder The folder to create.
on_create Called when the folder is created.
on_error Called if there is an error. A message is passed in.

Create_Path_From_Name(file, dirs)


Creates a folder from a file path.
file The file to create the folder from.
dirs A hash of created directories.

Create_Path_From_URL(file, dirs, on_create)


Creates a path from a URL.
file The file URL.
dirs A hash of created directories.
on_create Called when the path is created or not.

Upload_File_From_List(files, index, dirs, on_upload)


Uploads a file from a list.
files The list of files to upload.
index The index of the file to upload.
dirs The list of created folders.
on_upload Called when the files have been uploaded.

Download_File_From_List(files, index, on_download)


Downloads a file from a list.
files The list of files to download.
index The index of the file to download.
on_download Called when all files have been downloaded.

Query_Files(folder, search, on_query, on_error)


Queries a list of files from the server.
folder The folder to look in.
search The search string.
on_query Called when the files are queried with the file list passed in.
on_error Called if there is an error. A message is passed in.

Get_File_And_Folder_List(folder, on_query)


Grabs a list of files and folders from the server.
folder The folder to look in.
on_query Called when all files and folders have been queried. File list is passed in.

Query_From_Folders(folders, index, on_query)


Queries file and folders from a folder list.
folders The list of folders.
index The index of the folder to query from.
on_query Called when the list has been queried. File list is passed in.

Set_Config(name)


Sets a new config file for the project.
name The name of the config file.
