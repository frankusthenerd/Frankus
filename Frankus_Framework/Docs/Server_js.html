cServer

constructor(name)


Creates a new server.
name The name of the server.

Read_File(file, response, params)


Reads a file from the server.
file The file to be processed.
response The response to be populated.
params Extra parameters passed to the file.
throws An error if the file type is not found in the MIME table.

Write_File(file, response, params)


Writes a file to the server.
file The file to write to the server.
response The server response object.
params The parameters passed to the file.

Load_Rewrite_Rules(name)


Loads rewrite rules from a file.
name The name of the file.
throws An error if the rewrite rules are not formatted correctly.

Get_URL_Rewrite(file)


Gets the rewrite of a URL.
file The file URL.
returns The URL rewrite or the file if not found.

Handle_Request(request, response)


Handles a request from the server.
request The request object that is passed in.
response The response object that is passed in.
throws An error if something is wrong.

End_Response(status, message, response)


Ends a server response.
status The status associated with response.
message The message.
response The server response.

Start()


Starts the server.

Stop(on_stop)


Stops the server.
on_stop Called when the server is stopped.

cE_Services

constructor(server)


Creates a new e-services object.
server The server object.

Zip(params, response)


Zips up a folder.
params The parameters for the zip routine.
response The server response.

Zip_Cpp_Project(params, response)


Zips up a C++ project folder.
params The parameters for the zip routine.
response The server response.

Zip_JavaScript_Project(params, response)


Zips up a C++ project folder.
params The parameters for the zip routine.
response The server response.

Zip_Editor(params, response)


Zips up the editor.
params The parameters for the zip routine.
response The server response.

Archive_Site(params, response)


Archives the whole site.
params The parameters for the zip routine.
response The server response.

Coder_Doc(params, response)


Runs the Coder Doc documentation generator.
params The parameters for Coder Doc.
response The server response.

Terminal(params, response)


Runs a terminal command.
params The parameters for the terminal.
response The server response.

Create_Folder(params, response)


Creates a new folder if one does not exist.
params The parameter object with the passed in folder.
response The response object.

Query_Files(params, response)


Queries a group of files in a directory or a group of folders.
params The parameter object containing the folder and search.
response The server response.

Call(name, params, response)


Calls a service by name.
name The name of the service to call.
params The parsed server parameters.
response The server response.
throws An error if the service does not exist.
