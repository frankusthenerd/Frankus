constructor()


Creates a new binary tree.

Add(data, node)


Adds data to a node which is empty.
data The data to add.
node The node to add data to. This optional, defaults to root.

Find_Data(data, node)


Tries to find data in node.
data The data to find.
node The node to find the data in. This optional, defaults to root.
returns True if the data was found, false otherwise.

Clear()


Clears out a binary tree.
